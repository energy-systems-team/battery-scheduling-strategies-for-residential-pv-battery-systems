# Battery scheduling strategies for residential PV battery systems
This repository contains the codes and data used in publication "The impact of battery scheduling strategies on profitability considering battery usage costs and dynamic electricity pricing in Nordic residential PV-battery system", published in the Journal of Energy Storage (https://doi.org/10.1016/j.est.2026.121301). The structure of the repository should be as follows in order for the codes to run. In main.py, an example call is shown how to run the simulations, and in Visualizing_results_to_share.ipynb, the extraction of plots from the simulation results are shown.

The reinforcement learning models (RL and RL-BC strategies) cannot be run in the main.py files, but must be run separately.
